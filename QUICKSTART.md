## 🎮 立即開始遊戲

### 方法1：本地測試（推薦）

1. **確認服務器正在運行：**
   ```bash
   # 檢查是否有Python服務器在運行
   ps aux | grep "python.*8080"
   ```

2. **如果沒有運行，啟動服務器：**
   ```bash
   cd /Volumes/T7/pong_val
   python3 -m http.server 8080
   ```

3. **打開瀏覽器：**
   ```
   http://localhost:8080
   ```

### 方法2：直接雙擊打開

在 Finder 中找到 `index.html`，雙擊用瀏覽器打開（Safari/Chrome/Firefox都可以）

## 🎯 第一次遊玩流程

### 1. 開場（Landing Screen）
打開遊戲後，你會看到：
- 挑戰宣言：「這是一份專屬於你的挑戰，你準備好贏走我的心了嗎？」
- 一個大大的"解鎖 ❤️"按鈕

**操作：** 點擊"解鎖"按鈕

---

### 2. 進入遊戲
- 右上角會出現 **Token顯示** (💰 圖標)
- 左上角會出現 **收藏按鈕** (📚)
- 中間是 Pong 遊戲區域
- 有兩個模式可選：
  - **1 Player vs AI**（單人模式，推薦）
  - **2 Player Local**（雙人本地對戰）

**操作：** 點擊"Start Game ❤️"開始遊戲

---

### 3. 賺取 Tokens
- 用 **W/S 鍵** 控制你的擋板（左邊）
- 每贏一球 = **+1 Token 💰**
- 高速反彈（球速 ≥ 1.5x）= **+2 Tokens**
- Token會實時顯示在右上角

**目標：** 賺到 **5個Token** 就可以抽獎！

---

### 4. 第一次抽獎
當你有5個Token時：
- 右上角的"抽獎"按鈕會變成可點擊狀態
- **操作：** 點擊"抽獎 (5 💰)"按鈕

---

### 5. 扭蛋機體驗
- 畫面會切換到扭蛋機介面
- 中間是一個大顯示區域
- **操作：** 點擊"抽一次 (5 💰)"按鈕

**會發生什麼：**
1. 扭蛋機震動（動畫）
2. 顯示你抽到的卡片
3. 卡片會顯示稀有度（R/SR/SSR/UR）和顏色：
   - R: 銀灰色
   - SR: 寶藍色
   - SSR: 紫色
   - UR: 金色 ⭐

---

### 6. 收藏冊
隨時可以點擊左上角的"收藏 📚"按鈕查看：
- 所有已解鎖的卡片（彩色）
- 所有未解鎖的卡片（灰色，顯示"?"）
- 收集進度（例如：3/10, 30%）

---

### 7. 終極大獎（UR卡）
當你抽到UR卡時：
1. 扭蛋機顯示金色的UR卡
2. 2秒後自動跳轉到 **全屏大獎畫面**
3. 顯示：
   - 💝 浪漫的告白訊息
   - 🎁 真實世界的寶藏線索
4. 你可以點擊"繼續收集卡片"回到遊戲

**UR卡抽取機率：**
- 正常機率：5%
- 保底機制：最多5抽必定獲得UR（如果之前沒抽到）

## 💡 遊戲技巧

### 快速賺Token
1. **專注防守：** 讓球回彈就能得分
2. **等待高速球：** 當球速顯示加速後（遊戲每10秒+10%），成功反彈能得到雙倍Token
3. **善用暫停：** 累了可以隨時暫停，Token不會消失

### 快速收集所有卡片
- 總共10張卡片
- 平均需要抽 15-20 次才能收集全部
- 也就是需要贏 75-100 球左右
- 保底機制確保你一定能拿到UR

### 數據永久保存
- 關閉網頁也沒關係
- 你的Token、已解鎖的卡片都會保存
- 可以分多次遊玩慢慢收集

## 🎨 添加你的圖片（可選）

### 最簡單的方式：

1. **創建卡片文件夾：**
   ```bash
   mkdir -p assets/cards
   ```

2. **把你畫的圖片放進去：**
   - `assets/cards/UR1.png` - 最重要！
   - `assets/cards/SSR1.png`, `SSR2.png`
   - `assets/cards/SR1.png`, `SR2.png`, `SR3.png`
   - `assets/cards/R1.png`, `R2.png`, `R3.png`, `R4.png`

3. **刷新網頁：**
   - 遊戲會自動加載你的圖片
   - 如果圖片不存在，會顯示彩色佔位圖

**詳細繪製指南：** 查看 [ARTWORK_GUIDE.md](ARTWORK_GUIDE.md)

## 📞 遇到問題？

### 遊戲打不開
- 確認服務器正在運行（看上面的命令）
- 嘗試換一個瀏覽器（Chrome推薦）
- 檢查控制台（F12）看有沒有錯誤訊息

### 圖片顯示不出來
- 確認文件名正確（大小寫敏感！）
- 確認圖片在 `assets/cards/` 文件夾內
- 沒有圖片的話會顯示彩色佔位圖（這是正常的）

### Token沒有保存
- 確認瀏覽器不是隱私模式
- 某些瀏覽器設置可能會阻止localStorage
